<Aside type="note" title="Pre-1.0">
	agentsync is pre-1.0. Per [semantic versioning](https://semver.org/), the
	**interfaces** may still change in breaking ways before `v1.0.0` — both the
	programmatic API (Go package surface) and the CLI surface (commands and
	flags). The **core functionality** is considered **stable**: the three-state
	model, the canonical `~/.agentsync/` schema, and the apply/reconcile behavior
	are solid and safe to rely on. We avoid gratuitous interface churn and call
	out any breaking change in the [changelog](https://github.com/spxrogers/agentsync/blob/main/CHANGELOG.md).
</Aside>

## Package managers (recommended)

<Tabs syncKey="os">
	<TabItem label="macOS" icon="apple">
		```bash
		brew tap spxrogers/tap
		brew install agentsync
		```
	</TabItem>
	<TabItem label="Windows" icon="seti:windows">
		Scoop:

		```powershell
		scoop bucket add spxrogers https://github.com/spxrogers/scoop-bucket
		scoop install agentsync
		```

		Chocolatey:

		```powershell
		choco install agentsync
		```
	</TabItem>
	<TabItem label="Linux" icon="linux">
		Pick the package for your architecture (`amd64` or `arm64`).

		Debian / Ubuntu:

		```bash
		curl -fsSL https://github.com/spxrogers/agentsync/releases/latest/download/agentsync_linux_amd64.deb -o agentsync.deb
		sudo dpkg -i agentsync.deb
		```

		RPM:

		```bash
		sudo rpm -i https://github.com/spxrogers/agentsync/releases/latest/download/agentsync_linux_amd64.rpm
		```

		Arch (AUR) — wired but not published yet
		([issue #13](https://github.com/spxrogers/agentsync/issues/13)); use the
		`.deb`/`.rpm` or prebuilt binary for now:

		```bash
		yay -S agentsync-bin
		```
	</TabItem>
</Tabs>

## From source

Prefer to build it yourself, or want the latest unreleased commit?

```bash
go install github.com/spxrogers/agentsync/cmd/agentsync@latest
```

Or clone and build:

```bash
git clone https://github.com/spxrogers/agentsync
cd agentsync
go build ./cmd/agentsync
```

You'll need Go ≥ 1.26.2 (the version pinned in `go.mod`).

## Verify

```bash
agentsync --version
```

## Run a health check

Once installed, `doctor` confirms your environment is ready — PATH, home and
state writability, config schema, and the secrets backend:

```bash
agentsync doctor
```

<Steps>

1. **Install** the binary (above).

2. **Initialize** your config directory:

   ```bash
   agentsync init
   ```

3. **Continue** to [your first sync](/getting-started/first-sync/).

</Steps>
